RADIUS which stands for Remote Authentication Dial In User Service, is a network protocol – a system that defines rules and conventions for communication between network devices for remote user authentication and accounting. Commonly used by Internet Service Providers (ISPs), cellular network providers, and corporate and educational networks.

we all are using username and password to access the website/application on the internet. These pieces of information are known as principle and credentials respectively.Basically radius network uses the principle and credentials, along with third piece of information called as context, which gives a frame of reference of each login.

The principle, credentials, and the context are used together by a network security system to identify the user(authentication), to allow or deny access to network privileges(authorization) and to track usage of parameters(accounting).

                        

The RADIUS protocol serves three primary functions,

Authentication :

                                                               

Authentication allows a Radius server to grant or denty network accessed based on the information provided by the client. If everything is checks out, the server grants the client access to the network.

Authorization:

                         

Authorization determines what an authenticated client can and can not do an Radius server. Thease rules can limit what parts of a network can be accessed, what servers/application can be used, the length of time allowed on a network, and what type of network access is allowed.

Accounting:

                                                                                                

Accounting monitors and records the client's use of the network. Radius accounting tracks network usage, ensures the system doesn't overload and clients are billed appropriately.

 

FreeRadius Installation and Configuration:

We’ll explore the steps to install FreeRadius on Ubuntu environment with minimal configuration. These installation and configuration all related to RADIUS 3+ version

FreeRadius Installation on Ubuntu:

# apt-get install install freeradius

FreeRadius Configuration:

Main configuration file of FreeRadius under /etc/freeradius/3.0/* where we add or modify all Radius configuration. Add or modify necessary configuration in the following files.

  • /etc/freeradius/3.0/clients.conf
  • /etc/freeradius/3.0/users

Create Radius user:

Create Radius plain text user entry in /etc/freeradius/3.0/users file with the reference of sample configuration below.

Syntax :  <username> Cleartext-Password  := "<password>"   

e.g) testing Cleartext-Password  := "password"

Create Radius Client:

Create Radius plain text client entry in /etc/freeradius/3.0/clients.conf file with the reference of sample configuration below.

Syntax :

client  <IP address /DNS name>{
     secret      = <password>
}

e.g)

client 10.71.33.111{
     secret      = novell
}

Validate Radius Configuration:

# /usr/sbin/freeradius -d /etc/freeradius/ -CX

Start and enable Radius Service:

	# systemctl start|stop|restart|status freeradius or service freeradius start|stop|restart|status or 
	/etc/init.d/freeradius start|stop|restart

FreeRadius Troubleshooting:

# /usr/sbin/freeradius -X
